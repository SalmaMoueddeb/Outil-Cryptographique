
# Application Crypto

Ce projet contient deux fichiers principaux :

- `crypto_app.py` : L'application principale, une application web Streamlit pour des fonctionnalités liées aux cryptomonnaies.
- `cryptography_tool.py` : Un module d'outils pour des opérations ou utilitaires cryptographiques.

## Fonctionnalités
- Outils et utilitaires pour les cryptomonnaies
- Interface conviviale (basée sur Streamlit)
- Structure de code modulaire

## Prise en main

### Prérequis
- Python 3.8+
- [Streamlit](https://streamlit.io/)
- Toutes autres dépendances listées dans votre code (voir ci-dessous pour l'installation)

### Installation
1. Clonez ce dépôt :
   ```bash
   git clone <lien-du-repo>
   cd Crypto
   ```
2. (Optionnel) Créez et activez un environnement virtuel :
   ```bash
   python -m venv venv
   venv\Scripts\activate  # Sous Windows
   source venv/bin/activate  # Sous macOS/Linux
   ```
3. Installez les dépendances :
   ```bash
   pip install streamlit
   ```

### Lancement de l'application
Pour démarrer l'application Streamlit :
```bash
streamlit run crypto_app.py
```

### Utilisation
- Ouvrez votre navigateur à l'URL locale Streamlit indiquée dans le terminal.
- Interagissez avec l'application selon vos besoins.
